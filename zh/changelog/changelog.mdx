<Update label="03/30/2026">
  ## **文档中心、通讯录与 Viewer 更新**

  本次更新还包括文档中心、联系人管理、邀请流程以及文件预览能力的优化。

  ### **文档中心** <Badge color="blue" size="lg">新增</Badge>

  CodeBanana 现已在首页和个人中心中新增独立的 **Documentation** 标签页。

  文档中心现已支持问题搜索与 Changelog 同步更新，帮助用户在同一处更便捷地查找产品信息并了解最新变化。

  ### **默认项目** <Badge color="blue" size="lg">新增</Badge>

  默认项目现已包含功能引导文件。

  这可以帮助用户在进入项目后更快理解产品的核心能力，并为后续探索与上手提供更清晰的起点。

  ### **通讯录** <Badge color="blue" size="lg">新增</Badge>

  **Contacts** 现已同时包含组织内部联系人与外部联系人，并补充了相应的联系人说明。

  这些更新提升了联系人信息的可见性，也让用户更容易理解哪些联系人属于组织内部，哪些联系人作为外部联系人进行管理。

  ### **邀请弹窗** <Badge color="blue" size="lg">新增</Badge>

  邀请弹窗现已支持直接邀请内部联系人与外部联系人。

  这移除了此前的限制，使协作关系的建立在不同沟通方式和团队场景下更加灵活。

  ### **Viewer 预览优化** <Badge color="yellow" size="lg">优化</Badge>

  Viewer 预览区域现已针对更多文档类型进行了优化。

  以下文件类型的预览支持与查看体验均已得到优化：

  - DOC / DOCX
  - PDF
  - CSV
  - Excel

  这些更新提升了文档可读性，并让产品内的文件查看体验在常见文件格式之间更加一致。
</Update>

<Update label="03/27/2026">
  ## **新用户引导与组织管理**

  CodeBanana 现已支持新用户引导，以及组织架构与权限管理功能。以上更新现已正式上线至生产环境：[www.codebanana.com](http://www.codebanana.com)。

  本次发布覆盖个人用户、团队用户与企业用户的引导流程，帮助不同类型用户更高效地完成设置，并进入适合自己的工作流。所有引导步骤均可跳过，后续也可在 Contacts 中继续完成。

  ### **个人引导** <Badge color="blue" size="lg">新增</Badge>

  个人用户现在可以在引导流程中选择并提交自己的预期使用场景。

  这有助于 CodeBanana 为用户提供更贴合需求的初始体验。

  ### **团队与企业引导** <Badge color="blue" size="lg">新增</Badge>

  CodeBanana 现已为团队与企业用户提供更完整的引导流程。

  当组织 Owner 注册时，系统会自动创建一个默认组织，且 Owner 可在引导过程中修改组织名称。团队还可以通过 Excel 文件上传组织架构数据，包括部门与成员信息，用于支持批量邀请、注册以及团队结构初始化。

  组织成员也可以在引导流程中直接搜索并加入目标组织。所有引导操作均为可选项，支持跳过，后续可在 Studio 页面下的 Contacts 中继续补充与编辑。

  ### **组织架构上传与管理** <Badge color="blue" size="lg">新增</Badge>

  组织架构上传与管理现已支持更完整的管理员与成员管理流程。

  组织 Owner 现在可以设置多个组织管理员。组织 Owner 与管理员均可编辑成员角色与部门信息，并支持创建一级部门和子部门。

  当前还支持以下管理能力：

  - 单个或批量邀请组织成员
  - 查看邀请记录与邀请进度
  - 审核并通过组织加入申请
  - 批量上传与更新组织架构数据
  - 移除组织成员
  - 转移离职成员的项目，并指定新的接收 Owner

  这些更新帮助团队在同一处更高效地完成组织初始化配置与后续日常管理工作。
</Update>

  ## Team Plan、Agent 与工作空间体验更新

  本次生产环境更新主要覆盖三个方向，包含多项功能与产品体验优化：

  ### **Team Plan 更新**

  CodeBanana 现已提供更完整的 Team Plan 工作流，包括购买、成员管理和用量限制控制。

  这些更新让团队能够在工作空间内更清晰地管理套餐权限、分配成员并控制配额。

  ### **Agent 能力升级**

  本次发布扩展了 CodeBanana 在项目创建、配置以及外部集成方面的 Agent 能力。

  更新包括：

  - 支持 Claude Code Agent
  - 支持 Agent Config 配置
  - 支持 Skills Market
  - 支持 IM Bot 集成
  - 创建新项目时支持选择 Agent 模板

  这些变更让团队在不同项目场景下配置 Agent 与复用工作流时更加灵活。

  ### **UI、项目与开发者体验优化**

  本次发布还包括产品界面、项目文件管理以及开发者工具组织方式的更新。

  这些优化旨在让导航结构更加清晰、文件操作更加顺畅，并为在项目内工作的技术用户提供更一体化的使用体验。

## Project and chat relationship

Each project in CodeBanana has a corresponding group chat that supports three distinct chat modes for different collaboration needs.

### Chat modes

Every project group chat includes three modes:

#### Team agent

Team agent mode enables collaborative work with AI assistance:

- Only one team member can interact with the agent at a time
- Agent permissions are controlled by the group owner
- The owner can revoke agent access from other members during their conversation
- When multiple members request agent access simultaneously, the owner can only approve one request at a time
- Other requests remain in pending status until approved

#### My agent

My agent mode provides personal AI assistance:

- Primarily operates in "ask" mode for questions and planning
- Allows you to work on other tasks or ask questions while team agent is in use
- Personal workspace that doesn't interfere with team agent sessions

#### Discussion

Discussion mode is an open forum for all team members:

- All members can participate freely
- No agent involvement
- Standard group chat for team communication

## Adding members

You can expand your project team in two ways:

### Add organization members

Click to add existing members from your organization to the project group chat.

### Invite friends

Use the **Invite friends** feature to bring in external collaborators:

- Invited members join the current group chat directly
- They do NOT automatically join your organization
- This is useful for temporary collaborators or external partners

### Joining organizations

Organization membership is always user-initiated:

- Users must actively click **Join organization** from the top-left menu or Contacts directory
- This ensures intentional organization membership rather than automatic enrollment

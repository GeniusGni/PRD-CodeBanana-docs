New users complete a two-step onboarding flow to personalize their workspace. These selections are directly aligned with project template categories.

<Steps>
  <Step title="Choose usage type">

    - **Team / Enterprise** — collaborative and organizational usage

    <Frame>

    </Frame>
  </Step>
  <Step title="Choose use case">
    Select your primary purpose, which maps directly to **project template categories**, ensuring a consistent experience when creating new projects.

    <Frame>
      ![Onboardingchoosepurpose](/images/onboardingchoosepurpose.png)
    </Frame>
  </Step>
</Steps>

## **Organization Onboarding**

For team and enterprise users, onboarding includes organization setup or joining an existing organization.

### Admin actions

<Steps>
  <Step title="Create new org">
    Choose how you plan to use CodeBanana:

    - **Create New Organization**
    - **Join a Team**

    <Frame>
      ![Onboardingchooseteam](/images/onboardingchooseteam.png)
    </Frame>
  </Step>
  <Step title="Edit Team information">
    1. Enter organization name
       - Default is generated from admin email
       - Can be modified
    2. Configure:
       - Roles (Owner / Admin)
       - Team size

    <Frame>
      ![Onboardingeditteam](/images/onboardingeditteam.png)
    </Frame>
  </Step>
  <Step title="Import Organization Structure">
    Upload organization data through excel templates we provide:

    - Department structure
    - Member list

    <Frame>
      ![Onboardingteaminfo](/images/onboardingteaminfo.png)
    </Frame>
  </Step>
  <Step title="Confirmed">
    Before importing, you can use **Structure Preview** to verify:

    - Department hierarchy
    - Member assignments

    <Frame>
      ![Onboardingpreviewteam](/images/onboardingpreviewte.png)
    </Frame>
  </Step>
</Steps>

After upload:

- Invitation emails are automatically sent
- Members can register and join directly

This step is optional and can be skipped and you can manage structure later in **Contacts** page.

### Member actions

<Steps>
  <Step title="Join an organization">
    Choose to join an existing organization

    <Frame>
      ![Onboardingjointeam](/images/onboardingjointeam.png)
    </Frame>
  </Step>
  <Step title="Search an organization">
    - Search organization by name and choose from search results
    - Submit a join request

    <Frame>
      ![Onboardingsendrequest](/images/onboardingsendrequest.png)
    </Frame>
  </Step>
  <Step title="Wait for permission">
    After approval by the organization owner:

    - You are automatically added as a member
    - Your workspace switches to the selected organization
  </Step>
</Steps>

## **Onboarding Enhancements**

CodeBanana provides several built-in onboarding enhancements to help new users get started more smoothly.

### **CB-FAQ Skill**

Every project comes with a built-in **CB-FAQ** Skill.

This Skill stays in sync with product updates, allowing users to ask the agent direct questions about how to use CodeBanana and get product-related guidance inside the project.

<Frame>
  ![0418 Onboarding Faq](/images/0418-onboarding-faq.png)
</Frame>

### **Org Admin Workspace**

CodeBanana provides a dedicated **Org Admin Workspace** for organization owners and admins.

This workspace is designed as an intelligent operating console for organization management, usage analysis, and AI maturity insights.

**Entry path**:

- New users may receive this workspace by default during onboarding
- Existing users can create it from **New Project → Template → Data & Analyst**

<Frame>
  ![0418 Onboarding Orgadmin1](/images/0418-onboarding-orgadmin1.png)
</Frame>

<Frame>
  ![0418 Onboarding Orgadmin2](/images/0418-onboarding-orgadmin2.png)
</Frame>

**Permission note**:

- All queries and actions are validated against organization roles and structure
- Only **organization Owners** and **Admins** can access these management capabilities

**Supported capabilities**:

- **Organization structure setup and import/export**\
  Describe the organization in natural language, let the system structure it, preview the result, and confirm import. Excel import and export are also supported.
- **Department management**\
  Browse the organization tree, and create, rename, or delete departments. All write actions require confirmation.
- **Member management**\
  View members by department, assign or update departments, edit member information, and remove members.
- **Invitation management**\
  Invite new members, and view or cancel pending invitations.
- **AI operations metrics**\
  Query usage data such as token consumption, active users, and department rankings in natural language.
- **Daily AI maturity recommendations**\
  Enable automatic daily analysis to generate improvement suggestions and update the dashboard.
- **Manual AI maturity recommendations**\
  Generate AI maturity suggestions on demand for any selected time range.
